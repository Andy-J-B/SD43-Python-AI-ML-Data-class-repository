# Projects

- **project1** — BFS, DFS, and Uniform-Cost Search pathfinding algorithms on a 2D grid, with student template files and matching test suites.
- **project2** — Tic-Tac-Toe game with a minimax AI opponent, available as a complete CLI solution and starter templates for both CLI and GUI (Tkinter) versions.
- **project4** — Game AI implementations (minimax, alpha-beta pruning, expectimax) as student skeleton files with TODOs to complete.
- **project5** — Command-line 2048 game with an Expectimax AI; starter code with stub functions for students to implement.
- **project6** — Connect Four game with AI logic; template file with TODO sections for students to complete the gameplay and decision-making code.
- **project7** — MNIST handwritten digit classifier using a small convolutional neural network in PyTorch, with a student template and a main inference script.
- **project8** — Binary image classifier (cats vs. dogs) built from scratch with NumPy, including student-facing data preparation, training, and prediction scripts.
- **project9** — Q-learning implementation for a 5×5 Frozen-Lake-like gridworld; student template with step-by-step TODOs.
- **project10** — Pong game with a Q-learning AI agent trained via reinforcement learning over multiple episodes, including training, gameplay, and a saved model.
